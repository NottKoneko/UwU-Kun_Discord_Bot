# UwU-Kun Discord Bot

UwU-Kun is a fun and customizable Discord bot that includes AI-based features, logging, and moderation capabilities. This bot is designed to add a playful touch to your server while maintaining practical utilities.

## Features
- **AI Integration**: Chat with AI-based responses.
- **Logging**: Keep track of important events in your server.
- **Custom Commands**: Easily add and manage commands.
- **Moderation**: (More commands coming soon!)

## Future Plans
- **Moderation Commands**: I'm working on adding more comprehensive moderation tools (bans, mutes, etc.).
- **Customization**: Planned options to allow server owners to customize the bot's behavior to fit their needs.
- **Website Dashboard**: A future update will include a web-based dashboard for managing and monitoring the bot remotely.

## Prerequisites
- Python 3.8+
- Discord.py (`pip install discord.py`)
- AI libraries (optional)
